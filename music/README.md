## QQ音乐web APP
   实现了歌单推荐、歌手详情、歌单列表播放、歌曲播放和搜索、模式切换、歌词展示等功能
   通过jsonp和代理抓取真实QQ数据接口，需在联网下使用
   在不同移动端浏览器上有一些兼容性问题，还有一些本地存储的BUG，待后需优化完善
